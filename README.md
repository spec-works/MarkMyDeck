# MarkMyDeck

Software component for converting CommonMark/GitHub Flavored Markdown to Microsoft PowerPoint (.pptx) presentations.

## Specifications

Implements [CommonMark 0.31.2](https://spec.commonmark.org/0.31.2/) and [Office Open XML (ECMA-376)](https://www.ecma-international.org/publications-and-standards/standards/ecma-376/).

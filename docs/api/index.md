# API Reference

Browse the API documentation for the MarkMyDeck library and CLI tool.

## Namespaces

- [`MarkMyDeck`](MarkMyDeck.html) - Core conversion API (`MarkdownConverter`)
- [`MarkMyDeck.Configuration`](MarkMyDeck.Configuration.html) - Conversion options, themes, and styles
- [`MarkMyDeck.Converters`](MarkMyDeck.Converters.html) - Markdown-to-OpenXML rendering pipeline
- [`MarkMyDeck.Converters.BlockRenderers`](MarkMyDeck.Converters.BlockRenderers.html) - Block-level element renderers
- [`MarkMyDeck.Converters.InlineRenderers`](MarkMyDeck.Converters.InlineRenderers.html) - Inline element renderers
- [`MarkMyDeck.OpenXml`](MarkMyDeck.OpenXml.html) - Presentation building and slide management
- [`MarkMyDeck.SyntaxHighlighting`](MarkMyDeck.SyntaxHighlighting.html) - Code syntax highlighting
- [`MarkMyDeck.CLI.Commands`](MarkMyDeck.CLI.Commands.html) - CLI command definitions
